Implementation:
Create or update:
- modules/jail_enforcement.py

Add:

def is_user_jailed(user_id: str) -> bool:
    return active jail sentence exists and end_ts > now and status == "active"

def get_jail_remaining(user_id: str) -> int:
    return max(0, end_ts - now)

def get_jail_status_message(user_id: str) -> str:
    return "🚨 You’re jailed for Xm Ys more. Type !bail to leave early."

In main.py:
After parsing cmd and before teleport command handlers, check:

    return

Also block AI teleport:
In AI command executor / AI local command router:

    refuse with jail remaining message
    do not execute teleport

Do not block:
- !jailstatus
- !jailtime
- !bail

- normal chat
