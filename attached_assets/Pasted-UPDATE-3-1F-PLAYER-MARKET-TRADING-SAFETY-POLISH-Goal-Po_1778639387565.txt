PART 6 — SELL BADGE SAFETY
==================================================

Command:
!sellbadge [badge] [price]

Expected success:

✅ Badge listed!
👑 Crown — 20,000c
Listing ID: #12

Validation:
- seller must own badge
- badge must not be bound
- badge must be sellable
- badge must be tradable
- badge must not already be listed
- badge must not be staff-created
- badge must not be staff-duplicated
- price must be positive integer
- price must not be zero
- price must not be negative
- price must not be decimal
- seller must not be a bot

If invalid price:

⚠️ Price must be a positive number.

If badge is bound:

⚠️ This badge is bound and cannot be sold.

If already listed:

⚠️ Badge already listed.
Use !cancelbadge [id] first.

If badge not owned:

⚠️ You do not own that badge.

Rules:
- Listing should lock the badge.
- Listed badge should not be equippable unless listing is cancelled.
- If badge is equipped, unequip it before listing.
- Do not duplicate badge.
- Do not transfer badge during listing.
- Use transaction if possible.

PART 7 — BUY LISTING SAFETY
==================================================

Command:
!buybadge listing [listing_id]

Expected success:

✅ Badge bought!
👑 Crown from @Seller
Price: 20,000c

Seller message:

💰 Your badge sold!
👑 Crown — 20,000c

Validation:
- listing exists
- listing is active
- badge still belongs to seller
- badge is not bound
- badge is not staff-created
- buyer is not seller
- buyer is not bot
- buyer has enough coins
- listing price is valid

Failure messages:

⚠️ Listing not found.

⚠️ This listing is no longer available.

⚠️ Not enough coins.

⚠️ You cannot buy your own listing.

Rules:
- Deduct buyer coins once.
- Credit seller coins once.
- Transfer badge ownership once.
- Mark listing sold once.
- Set badge listed = 0.
- Ensure equipped = 0 after transfer.
- Use DB transaction.
- Prevent race condition if two buyers buy same listing at same time.
- Only one buyer should succeed.

PART 9 — TRADE REQUEST SYSTEM
==================================================

Add only if safe. If risky, skip and do not show in help.

Purpose:
Let players trade badges safely with confirmation.

Commands:
!trade @user
!tradeadd [badge]
!tradecoins [amount]
!tradeview
!tradeconfirm
!tradecancel

Basic flow:
1. Player A: !trade @PlayerB
2. Player A adds badge/coins.
3. Player B adds badge/coins.
4. Both must type !tradeconfirm.
5. System swaps items/coins atomically.

Do not allow instant one-sided trade.

Validation:
- target is in room or known player if safe
- target is not bot

- timeout trade after 5 minutes of inactivity

Skip trading system and keep badge market only.
