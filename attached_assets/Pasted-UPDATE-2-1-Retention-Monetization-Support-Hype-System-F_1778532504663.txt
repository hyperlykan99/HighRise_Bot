10C — EXOTIC FULL ROOM HYPE EVENT
==================================================

When player gets Exotic ore/fish:

Action:
1. Temporarily pause games and economy actions.
2. Temporarily mute everyone except bots/staff if supported.
3. All bots send hype messages.
4. EmceeBot makes a main announcement.
5. Send mining/fishing notification to subscribed players.
6. After a short duration, unmute everyone.
7. Resume games and economy actions.
8. Log the event.

This should feel like a once-in-a-room-history moment.

Exotic event flow:

Step 1:
Detect Exotic ore/fish.

Step 2:
Activate hype lock:
- Pause mining
- Pause fishing
- Pause casino games
- Pause blackjack
- Pause poker
- Pause economy payouts if needed
- Pause auto events if needed

Step 3:
Mute room briefly if API supports it:
- Mute normal players only
- Do not mute owner/admin/staff
- Do not mute bots
- Duration: 10–20 seconds max

Step 4:
Bots send hype messages.

Step 5:
Unmute everyone.

Step 6:
Resume paused systems.

Step 7:
Log hype event.

Example messages:

EmceeBot:
🚨🚨 EXOTIC FIND 🚨🚨
@Player just found Exotic [item_name]!
The room is witnessing history.

After hype:

EmceeBot:
✅ Hype break complete.
Everyone is unmuted.

During Exotic Hype Event:

  🚨 Exotic Hype Break active. Mining resumes shortly.

- !fish should temporarily reply:
  🚨 Exotic Hype Break active. Fishing resumes shortly.

- Blackjack/Poker should temporarily reply:
  🚨 Exotic Hype Break active. Games resume shortly.

- Gold/economy commands should either queue or pause safely.

Mute rules:
- Mute everyone only if Highrise API supports safe temporary muting.
- If room-wide mute is not supported, use soft mute:
  - pause command handling
  - ask players to watch the hype
  - do not attempt unsafe moderation actions

Never mute:
- owner
- admins
- managers
- moderators/staff
- bots

Duration:
10–20 seconds only.

Safety:
- Always unmute in finally block.

Emergency recovery command:

Owner/Admin only.

Behavior:
- Unmutes everyone if possible.
- Clears hype lock.
- Resumes mining/fishing/casino/events.
- Logs emergency unlock.

Response:
