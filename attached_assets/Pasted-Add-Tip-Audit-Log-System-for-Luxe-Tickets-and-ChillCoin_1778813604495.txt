Add Tip Audit Log System for Luxe Tickets and ChillCoins

Goal:
Log every player tip to any bot so owner can verify complaints like:
- “I tipped but didn’t receive Luxe Tickets”
- “I bought ChillCoins but didn’t receive coins”
- “The bot rewarded the wrong amount”
- “I tipped the wrong bot”

Do not change:
- jail system
- blackjack
- poker
- workflows
- bot startup
- coinpack pricing logic unless logging needs it

Part 1 — Create tip audit table

In database.py init_db(), create:

CREATE TABLE IF NOT EXISTS tip_audit_logs (
    id INTEGER PRIMARY KEY AUTOINCREMENT,
    event_hash TEXT,
    sender_user_id TEXT,
    sender_username TEXT,
    receiver_user_id TEXT,
    receiver_username TEXT,
    bot_mode TEXT DEFAULT '',
    raw_tip_type TEXT DEFAULT '',
    raw_tip_id TEXT DEFAULT '',
    gold_amount INTEGER DEFAULT 0,
    luxe_expected INTEGER DEFAULT 0,
    luxe_awarded INTEGER DEFAULT 0,
    luxe_balance_before INTEGER DEFAULT 0,
    luxe_balance_after INTEGER DEFAULT 0,
    coins_awarded INTEGER DEFAULT 0,
    coins_balance_before INTEGER DEFAULT 0,
    coins_balance_after INTEGER DEFAULT 0,
    status TEXT DEFAULT '',
    failure_reason TEXT DEFAULT '',
    duplicate_detected INTEGER DEFAULT 0,
    created_at TEXT DEFAULT (datetime('now'))
);

Add index:
CREATE INDEX IF NOT EXISTS idx_tip_audit_sender ON tip_audit_logs(sender_username, created_at);
CREATE INDEX IF NOT EXISTS idx_tip_audit_receiver ON tip_audit_logs(receiver_username, created_at);
CREATE INDEX IF NOT EXISTS idx_tip_audit_event_hash ON tip_audit_logs(event_hash);

Part 3 — Log every tip in modules/tips.py

In process_tip_event, log these cases:

A. Non-gold tip:
status = "ignored_non_gold"
gold_amount = 0
luxe_awarded = 0
failure_reason = "not_gold_tip"

B. Duplicate:
status = "duplicate_ignored"
duplicate_detected = 1
gold_amount = gold
luxe_awarded = 0

C. Luxe reward success:
status = "success_luxe"
gold_amount = gold
luxe_expected = gold
luxe_awarded = gold
luxe_balance_before = before
luxe_balance_after = after
coins_awarded = 0

D. Luxe reward failed:
status = "failed_luxe"
gold_amount = gold
luxe_expected = gold
luxe_awarded = 0
failure_reason = repr(error)

Important:
1g = 1 🎫 Luxe Ticket.
No bonus.
No ChillCoins from tips.
