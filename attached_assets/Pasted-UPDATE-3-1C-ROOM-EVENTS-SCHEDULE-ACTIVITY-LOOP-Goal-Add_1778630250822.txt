PART 7 — LOW-SPAM ACTIVITY LOOP
==================================================

If an activity loop exists, polish it.
If not, add only a simple safe loop if the project already has background task helpers.

Activity loop purpose:
Occasionally remind players what they can do.

Suggested interval:
15–25 minutes.

Random prompts:

🎮 Looking for something to do? Try !mine, !fish, !help games, or !tele list.

🎣 Fishing break! Type !fish or !autofish to start.

⛏️ Mining time! Type !mine or !automine to earn coins.

🃏 Casino tables are open. Blackjack: !bet [amount]. Poker: !poker.

🎁 Don’t forget your daily reward: !daily.

📅 Want events? Type !events or !nextevent.

Rules:
- Max one activity prompt per interval.
- Do not prompt if room is empty.
- Do not prompt if bot is in stability mode ON.
- Do not prompt during command spam.
- Do not prompt more than 4 times per hour.
- Do not start duplicate loops after reconnect.
- Do not send to DMs. Public only.
- Keep under 220 characters.
