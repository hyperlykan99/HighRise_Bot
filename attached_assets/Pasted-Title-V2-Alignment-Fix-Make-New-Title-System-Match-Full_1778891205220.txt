- notification subscription system

Only modify:
- modules/shop.py if titles stay there
- OR create modules/title_system.py if cleaner
- database.py only with safe CREATE TABLE IF NOT EXISTS migrations
- main.py only for routing
- help_cmds.py only for help text
- fishing/mining/casino/daily modules only for stats/perk integration
