Current status:
Poker V2 core loop works now, but there are important bugs and polish changes needed.

Current bugs from live screenshots:
1. After winner announcement, the next hand does not always start cleanly.
2. !leave during/between hands is confused.
3. !join says player is seated, but !poker leave says player is not at table.
4. !pleave still goes to ChillTopiaMC unknown command.

6. Board/showdown must NOT run when everyone else folded or left.
7. Current player action messages are not detailed enough.
8. Player turn whisper needs better format and current hand ranking.
9. Cards should display 10, not T.

PART 3 — Add Hand-End Lock
==================================================

Add a hand cleanup lock:

_T["cleanup_in_progress"] = False
_T["hand_ended"] = False

When a winner is determined:
_T["hand_ended"] = True
_T["cleanup_in_progress"] = True

During cleanup:
- block actions
- block duplicate resolver calls
- block board run
- block showdown run
- allow !table and !hand only as safe read commands
- allow !leave but process it as between-hand leave if hand already ended

Action guard:
If cleanup_in_progress:
    whisper "Hand is settling. Please wait."
    return

Resolver:
If hand_ended is True:
    return immediately

This prevents:
- fold win followed by board run
- leave followed by all-in board run
- duplicate payouts

PART 4 — Fix Fold/Leave Winner Logic
==================================================

If only one non-folded player remains:
- award pot immediately
- DO NOT run flop/turn/river
- DO NOT run showdown
- call complete_hand(bot, reason="fold_win")

Correct message:

🏆 @Winner wins 150 coins.
Reason: Everyone else folded.

If a player leaves and folds:
- mark folded
- resolve
- if one player remains, winner gets pot
- no board run

If no eligible players remain:
- cancel/refund safely
- no board run

Important:
Board run/showdown is only allowed when:
- at least 2 eligible players remain
AND
- they are all-in or betting is complete to river

Never run board after everyone else folded.

==================================================

Add owner/admin-only commands.

Commands:

- Pause auto-starting new hands.
- Active hand may finish, but no next hand starts.
Message:
⏸️ Poker paused. Current hand may finish, but no new hand will start.

!poker resume
- Resume auto-start.
- If 2+ seated and no active hand, start countdown.
Message:
▶️ Poker resumed.

!poker close
- Close poker table safely.
- If no active hand: cash out all seated players and clear table.
- If active hand: stop timer, refund current hand contributions, cash out stacks, clear table.
Message:
🛑 Poker table closed. All players cashed out.

!poker refund
- Refund current active hand only.
- Return current hand contributions to player stacks.
- Clear current hand.
- Keep players seated.
Message:
💸 Current hand refunded. Players remain seated.

!poker refund @user
- Refund/cash out specific player if seated.
- If active hand and player is in hand, fold/remove safely.
Message:
💸 @Player refunded/cashed out.

!poker forcefinish
- Safely finish active hand.
- If one eligible player remains, award pot.
- If 2+ eligible players remain, go to showdown only if valid.
- Otherwise refund hand.

⚠️ Poker hand force-finished.
