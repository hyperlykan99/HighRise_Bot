PART 2 — EVENT SYSTEM OVERVIEW
==================================================

Create/polish room events that can be active, scheduled, announced, and shown in player commands.

Core event types:

Mining Rush
- +2 mining luck
- optional +1 mining speed
- affects !mine, !automine, !automine luxe

Fishing Frenzy
- +2 fishing luck
- optional +1 fishing speed
- affects !fish, !autofish, !autofish luxe

Double XP Hour
- 2x XP from mining, fishing, trivia, missions, games
- does not double ChillCoins unless explicitly configured

Collection Hunt
- +2 collection luck
- better chance for missing collection items if system supports it
- if missing-item targeting is risky, apply +2 general mining/fishing luck only

Casino Night
- bonus season points for casino activity
- no odds changes
- no payout changes

Trivia Rush
- bonus XP and season points for trivia
- no reward caps

Rules:
- Events should affect activity/boosts, not guarantee rare items.
- Events must appear in !luck / !mineluck / !fishluck if relevant.
- Events must not add HR Gold rewards.
- Events must not create reward caps.
