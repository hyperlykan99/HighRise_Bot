==================================================
PART 4 — STREAK LOGIC
==================================================

Streak rules:

If player claims for the first time:
streak = 1

If player claimed yesterday or within the valid streak window:
streak += 1

If player missed more than 1 day:
streak resets to 1

If player already claimed today:
no reward
streak unchanged

Recommended valid streak window:
- claim once per calendar day
- streak continues if previous claim was yesterday
- if using hours, allow 20 to 48 hours since last claim for forgiving streaks

Keep logic simple and safe.

==================================================
PART 5 — REWARD AMOUNTS
==================================================

Use safe small rewards.

Base reward:
100 coins

Streak bonus:
+25 coins every day after day 1

Max daily reward cap:
500 coins

Formula:
reward = min(100 + ((streak - 1) * 25), 500)

Examples:
Day 1 = 100c
Day 2 = 125c
Day 3 = 150c
Day 7 = 250c
Day 17+ = 500c cap

Do not make rewards too high.
Do not reward gold.
Do not affect VIP.
Do not affect casino odds.
Do not affect mining/fishing odds.

If there is already a daily reward value in config:
- use existing config if present
- otherwise use above defaults
