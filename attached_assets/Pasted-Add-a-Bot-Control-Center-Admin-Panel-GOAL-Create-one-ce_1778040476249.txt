Add a Bot Control Center / Admin Panel.

GOAL:
Create one central command hub for owners/admins/managers to view and manage all major bot systems:
- room utilities
- roles
- moderation
- economy
- bank
- shop
- badges/titles
- VIP
- casino
- poker
- mining
- events
- notifications
- bot modes/outfits
- welcome/intervals
- health/debug/audit

IMPORTANT:
- Do not remove existing features.
- Keep all existing commands working.
- Use SQLite.
- Keep every Highrise message under 250 characters.
- Split panels into pages.
- Use role-based visibility.
- Add explicit command routing before unknown-command fallback.
- Prevent silent failures.
- Every command must reply with success, usage, permission error, or clear error.

==================================================
COMMANDS
==================================================

Add:
- /control
- /control 1
- /control 2
- /control 3
- /control room
- /control economy
- /control casino
- /control games
- /control shop
- /control staff
- /control system
- /adminpanel
- /ownerpanel
- /managerpanel

Permissions:
- Owner sees everything.
- Admin sees most controls except owner-only commands.
- Manager sees room/events/casino/game controls.
- Moderator sees moderation/report controls only.

If no permission:
"Staff only."
