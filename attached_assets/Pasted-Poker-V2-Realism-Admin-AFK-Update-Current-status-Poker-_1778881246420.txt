PART 4 — Realistic Raise Rules
==================================================

Add realistic Texas Hold’em raise tracking.

Table fields:
_T["last_raise_amount"] = big_blind
_T["last_aggressor"] = None

At hand start:
last_raise_amount = big_blind
last_aggressor = big_blind player

When a player raises:
- previous_current_bet = _T["current_bet"]
- new_total_bet = amount from !raise
- raise_size = new_total_bet - previous_current_bet
- valid raise_size must be >= last_raise_amount unless player is all-in
- update last_raise_amount = raise_size only if full legal raise
- update last_aggressor = player only if full legal raise

Important:
!raise amount means total bet amount, not “add this much.”

Example:
Current bet: 100
Player types:
!raise 300

Meaning:
Player’s total bet becomes 300.
Raise size is 200.

Minimum next raise:
If last_raise_amount = 200, next minimum total raise is 500.

Invalid:
Minimum raise is 500 coins total.

Short all-in:
If player all-ins for less than minimum raise:
- allow all-in
- do NOT update last_raise_amount
- do NOT reopen action for players who already acted and matched the prior bet

Add logs:
[POKER V2 RAISE] user=@Player old_bet=100 new_bet=300 raise_size=200 min_raise=100 full_raise=true
[POKER V2 RAISE] user=@Player short_allin=true reopen_action=false

PART 11 — Admin Safety Commands
==================================================

Add owner/admin-only commands:

!poker pause
- Pause auto-starting new hands.
- Active hand may finish, but no next hand starts.
Message:
⏸️ Poker paused. Current hand may finish, but no new hand will start.

!poker resume
- Resume auto-start.
- If 2+ eligible seated players and no active hand, start countdown.
Message:
▶️ Poker resumed.

!poker close
- Close poker table safely.
- If no active hand: cash out all seated players and clear table.
- If active hand: stop timer, refund current hand contributions to stacks, cash out stacks, clear table.
Message:
🛑 Poker table closed. All players cashed out.

!poker refund
- Refund current active hand only.
- Return current hand contributions to player stacks.
- Clear current hand.
- Keep players seated.
Message:
💸 Current hand refunded. Players remain seated.

!poker refund @user
- Refund/cash out specific player if seated.
- If active hand and player is in hand, fold/remove safely.
Message:
💸 @Player refunded/cashed out.

!poker forcefinish
- Safely finish active hand.
- If one eligible player remains, award pot.
- If 2+ eligible players remain and board is complete, showdown.
- If 2+ eligible players remain and board incomplete, run board only if valid.
- Otherwise refund hand.
Message:
⚠️ Poker hand force-finished.

!poker status
- Admin table status summary.

Keep these hidden from normal player help.
