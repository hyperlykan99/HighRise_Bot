==================================================
PART 1 — Global Menu Style Rules
==================================================

All menus must be:

- clean
- short
- emoji-friendly
- easy to read on mobile
- under Highrise message limit
- split into multiple whispers if needed
- no giant walls of text
- no broken/unused commands shown
- no duplicate commands
- no outdated aliases
- no commands that show “unknown command”

Use this style:

Title line:
🎮 Games
🛍️ Shop
🎖️ Badges
🏷️ Titles
🔔 Notifications
💰 Economy
🧪 QA Tools

Command lines:
💰 Balance: !bal
🎁 Daily: !daily
🎖️ Badges: !badges
🏷️ Titles: !titles
🔔 Alerts: !notifysettings

Avoid long paragraphs.
