Add helper:

def _safe_get(entry, *names, default=None):
    for name in names:
        if hasattr(entry, name):
            return getattr(entry, name)
        if isinstance(entry, dict) and name in entry:
            return entry[name]
    return default

Use it in _check_cmd().
