PART 4 — BLACKJACK MATCHING CARD BONUS SYSTEM
==================================================

Add a matching-card bonus system.

Goal:
If a player gets matching starting cards, they can earn bonus chips.

Bonus should be paid separately from normal blackjack winnings.

Recommended matching bonus tiers:

1. Pair Bonus
Condition:
First 2 player cards have same rank.
Example: 8♠ 8♦

Reward:
Small bonus.

2. Color Pair Bonus
Condition:
Same rank and same color.
Example: 8♥ 8♦ or 8♣ 8♠

Reward:
Better bonus.

3. Suit Pair Bonus
Condition:
Same rank and same suit style if deck logic supports duplicate decks.
Example: 8♥ 8♥ in multi-deck shoe.

Reward:
Higher bonus.

4. Perfect Pair Bonus
Condition:
Exact same rank and suit in multi-deck shoe.
Example: K♠ K♠

Reward:
Highest bonus.

If suit-pair/perfect-pair distinction is too complex with current deck model, implement:
- Pair
- Same Color Pair
- Perfect Pair

==================================================

Current goal:
AutoMine and AutoFish should save sessions when bots restart.

Previous rule still applies:
- No offline mining/fishing.
- Stop if player leaves.
- Do not backpay missed attempts during downtime.

New behavior:
When bot restarts:
1. Load active AutoMine sessions.
2. Load active AutoFish sessions.
3. Check if player is still in the room.
4. If player is still present, resume safely with remaining limits.
5. If player is not present, mark session stopped.
6. Do not credit missed attempts during downtime.
7. Do not auto-restart if session expired during downtime.
8. Keep attempts/time limits.
