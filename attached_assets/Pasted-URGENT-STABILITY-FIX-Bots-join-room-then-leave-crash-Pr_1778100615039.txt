PART 2: CRASH-PROOF MULTI-BOT RUNNER
==================================================

Each bot client must run in its own protected async task.

Rules:
- If one bot crashes, log it.
- Do not kill the other bots.
- Restart only the crashed bot after delay.
- Use backoff: 5s, 10s, 30s, 60s max.
- Do not let asyncio.gather crash everything.
- Use return_exceptions=True or individual task wrappers.

Pseudo behavior:
run_bot_forever(bot_config):
    while True:
        try:
            await start_bot(bot_config)
        except Exception as e:
            log_crash(bot_config, e)
            mark bot restarting
            sleep backoff
            continue

Console:
"[CRASH] Poker bot crashed in startup: <error>"
"[RESTART] Poker bot restarting in 10s"
