Upgrade titles into Title V2:

- perks and boosts
- visible stats
- title progress
- owner force-give/remove/set tools
- loadouts

- prestige display

Do NOT modify:
- Poker V2 gameplay
- badge market, unless needed only for shared display/boost helpers
- jail

- notification subscription system
- unrelated games

Only modify:
- modules/shop.py if keeping titles there
- or create modules/title_system.py if cleaner
- fishing module only to apply/check fishing title stats/perks
- mining module only to apply/check mining title stats/perks

- database.py only if safe migrations are needed
- main.py only if routing is needed
- help_cmds.py only for title help/global help
