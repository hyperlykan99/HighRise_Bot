Rebuild the full role-based Help Command System.

Problem:
Many commands exist now, but the help menus are incomplete.
I want every working command placed into help, organized by role and module.

Goal:
Create a complete help system where:
- Players see only player commands.
- VIPs see player + VIP commands.
- Moderators see player + mod commands.
- Managers see player + mod + manager commands.
- Admins see player + mod + manager + admin commands.
- Owners see everything.
- Each bot/module help shows commands it owns.
- Help should never list commands that are not routed.
- Routed commands should have help entries unless hidden/internal.
- Everything must be paginated under 250 characters.

IMPORTANT:
- Do not remove any commands.
- Keep multi-bot command ownership.
- Keep Host, Banker, Blackjack, Poker, Miner, Shop, Security, DJ, Event bots.
- Keep all existing modules.
- Use the command registry as the source of truth.
- No command should say unknown if it is listed in help.
- No command should be hidden from owners unless internal/debug.
- Keep messages short and readable.
