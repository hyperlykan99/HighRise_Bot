Update ALL help menus, command menus, role menus, and module help for the entire Highrise bot.

IMPORTANT:
- Do not remove existing features.
- Do not change game logic unless needed for help routing.
- Keep every Highrise chat message under 250 characters.
- Split long menus into pages.
- Add explicit command routing before the unknown-command fallback.
- Prevent crashes from bad inputs.
- Normal players should not see staff-only commands unless the help menu says “Staff only.”
- Staff menus should only show commands the user’s role can access.

Current bot modules/features include:
- Mini games: trivia, scramble, riddle, answer timer, auto games
- Coin economy: daily, balance, leaderboard, XP, levels
- Shop: badges, titles, VIP, item info, equip, myitems
- Bank: send coins, transactions, bank settings, anti-abuse, bank notifications
- Casino: casual blackjack /bj
- Casino: realistic blackjack /rbj with shoe
- Casino control panel
- Blackjack crash/restart recovery
- Gold system: owner-only gold tips, refunds, gold rain, VIP/role/title/badge gold rain
- Events: manual events and auto events
- Quests and achievements
- Reputation system
- Reports and moderation
- Staff roles: owner, admin, manager, moderator
- Maintenance: backup, cleanup, softrestart, botstatus, dbstats, maintenance mode
