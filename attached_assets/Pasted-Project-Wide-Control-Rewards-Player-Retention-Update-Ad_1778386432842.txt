PART 6 — DAILY QUESTS
==================================================

Add command:

/daily
/dailies
/claimdaily

Owner:
EmceeBot or BankerBot

Recommended owner:
BankerBot for rewards
EmceeBot can announce completion if needed

Permission:
Everyone

Purpose:
Give players daily goals.

Daily quest examples:
- Mine 25 times = 5,000 coins
- Catch 15 fish = 3,000 coins
- Play 3 blackjack rounds = 2,000 coins
- Play 3 poker rounds = 2,000 coins
- Tip any bot 1g = 1,000 bonus coins
- Join First Find Race = 2,000 coins

Use existing activity logs if possible:
- mining records
- fish catch records
- blackjack rounds
- poker rounds
- bot tip logs
- first find race participation

If no logs exist for a quest, skip that quest or mark as unavailable.

Daily reset:
Every 24 hours based on server date.

Database:
daily_quests:
- id INTEGER PRIMARY KEY AUTOINCREMENT
- quest_key TEXT
- description TEXT
- reward_coins INTEGER
- enabled INTEGER
- created_at TEXT

daily_quest_progress:
- id INTEGER PRIMARY KEY AUTOINCREMENT
- user_id TEXT
- username TEXT
- quest_key TEXT
- progress INTEGER
- target INTEGER
- completed INTEGER
- claimed INTEGER
- quest_date TEXT
- updated_at TEXT

/daily example:

BankerBot:
📅 Daily Quests
1. Mine 25 times: 12/25 — Reward 5,000c
2. Catch 15 fish: 4/15 — Reward 3,000c
3. Play 3 BJ rounds: 0/3 — Reward 2,000c

Use /claimdaily when complete.

/claimdaily:
- pays completed unclaimed daily rewards
- prevents double claiming
- updates balance
- logs payout
