Part 3 — Realistic Hand Start Sequence

When a hand starts:

1. Lock table state.
2. Select active seated players.
3. Rotate dealer.
4. Assign small blind and big blind.
5. Shuffle deck.
6. Deal 2 private cards to every active player.
7. Save all hole cards to DB.
8. Post blinds.
9. Announce dealer/blinds publicly.
10. Wait 5 seconds.
11. Whisper cards to every player.
12. Guaranteed whisper to first actor again.
13. Publicly announce first actor turn.
14. Start turn timer.

No player action can be accepted until first actor card whisper succeeds or fails after retry.

If player acts too early:
⏳ Dealer is preparing the hand. Please wait.

Public blind message:
🃏 Hand #12
SB: @Player1 50 🪙 | BB: @Player2 100 🪙
Pot: 150 🪙

After 5 seconds:
Public:
⏳ @Player’s turn. 30s

Private to current player:
🎯 Your turn. Cards: A♠ K♦
Pot: 150 🪙
Use !check, !call, !raise 500, !fold, or !allin.

Part 4 — Fix First Player Missing Cards

First actor must receive cards twice:
1. Normal deal whisper.
2. Guaranteed turn whisper before their first action.

Use the same stored cards.
Do not redeal.
Do not reshuffle.

If cards are missing:
- pause the hand
- set phase to recovery_required
- do not let betting continue
- tell staff:
⚠️ Poker card delivery failed. Use !poker recoverystatus.

Add logs:
[POKER DEAL] user=@Player cards_saved=true
[POKER WHISPER] user=@Player cards_sent=true
[POKER FIRST ACTOR] user=@Player cards_resend=true
[POKER FIRST ACTOR] gate_open=true

Part 7 — 2-Player All-In Compatibility

If there are 2 players and:

- Player A goes all-in
- Player B calls
- no active player still needs to act

Then immediately run the board.

Do not wait for another turn.

Correct sequence:
🔥 All-in called! Running the board...
🃏 Flop: 4♠ 9♦ K♥
🃏 Turn: J♣
🃏 River: 2♥
👀 Showdown!

If all-in happens on flop:
🔥 All-in called! Running the board...
🃏 Turn: J♣
🃏 River: 2♥
👀 Showdown!

If all-in happens on turn:
🔥 All-in called! Running the board...
🃏 River: 2♥
👀 Showdown!

If all-in happens on river:
👀 Showdown!

Part 8 — Betting Resolver

After every poker action, call one resolver:

resolve_betting_state()

It decides:

1. If only one non-folded player remains:
   award pot immediately.

2. If no active players can act:
   deal remaining board and showdown.

3. If all active players have acted and matched current bet:
   advance to next street.

4. Otherwise:
   move to next player.

This must run after:
!check
!call
!raise
!fold
!allin

Part 9 — Side Pots

Add side pot calculation for realistic all-ins.

At showdown:
1. Read every player's total_contributed.
2. Sort contribution levels.
3. Build pot layers.
4. Each side pot has eligible players:
   - did not fold
   - contributed enough for that layer
5. Award each pot separately.
6. If tied, split that pot.

Example:
Main Pot: 3 players eligible
Side Pot: 2 players eligible

Showdown message:
🏆 Main Pot 3,000 🪙 → @Player1
🏆 Side Pot 5,000 🪙 → @Player2

Part 10 — Player Leaving Room

Add poker leave-room behavior.

If player leaves room before active hand:
- cash out full table stack to wallet
- remove from table

Public:
🚪 @Player left poker. Stack cashed out.

If player leaves during active hand and is not all-in:
- auto-fold them
- committed chips stay in pot
- remaining hand stack is returned after hand
- mark remove_after_hand=1

Public:
🚪 @Player left during the hand and folded. Committed chips stay in the pot.

If player leaves during active hand and is all-in:
- keep them eligible for showdown
- mark remove_after_hand=1
- after showdown, cash out remaining/winnings and remove from table

Public:
🚪 @Player left but is all-in. Their hand will run to showdown.

Part 11 — Anti-AFK

Add AFK strikes.

Rules:
1 timeout = warning
2 timeouts = warning
3 timeouts = removed from poker table

If timeout happens and player can check:
- auto-check
- add AFK strike

If timeout happens and player must call:
- auto-fold
- add AFK strike

Messages:
⏰ @Player timed out. AFK warning 1/3.
⏰ @Player timed out. AFK warning 2/3.
⏰ @Player removed from poker for AFK.

If removed during active hand:
- if not all-in: auto-fold and remove after hand
- if all-in: keep until showdown, then remove
