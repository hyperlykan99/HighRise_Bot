Add a complete realistic ore-based Mining grind game for room retention.

GOAL:
Create a unique mining/grinding game that keeps players active in the room.
Players mine realistic ores, minerals, and gemstones, upgrade pickaxes, sell ores, craft rewards, earn mining XP, and compete on leaderboards.

Theme:
Chill Lounge Mine.
Relaxing and welcoming for all players, but the materials should be realistic ores/minerals/gems.

IMPORTANT:
- Do not remove existing features.
- Keep coins, XP, event coins, bank, shop, badges, titles, VIP, casino, BJ, RBJ, poker, events, notifications, profiles, roles, admin tools, and all current systems.
- Use SQLite.
- Use the existing number formatter with commas and short values.
- Keep every Highrise message under 250 characters.
- Split long messages into pages.
- Add explicit command routing before unknown-command fallback.
- Prevent crashes from bad inputs.
- Mining must not give unlimited coins too fast.
- Log major rewards in economy ledger/admin logs if available.
- Commands must always reply with success, usage, permission error, cooldown, or clear error.
- No silent failures.

PART 2: CORE GAME LOOP
==================================================

When player uses:
/mine

The bot checks:
- mining_enabled
- player is in room if mining_requires_room = true
- cooldown
- energy
- mining level
- pickaxe/tool level
- active mining event bonus
- VIP mining bonus if any
- boost effects if active

Then the bot gives:
- random realistic ore/mineral/gem drop
- mining XP called MXP
- rare drop chance
- small coin bonus chance if enabled

Example messages:
"⛏️ You mined Iron Ore x2 | +15 MXP."
"⛏️ You mined Quartz x1 | +20 MXP."
"💎 @user found Diamond x1! +300 MXP."
"☄️ @user found a Meteorite Fragment! Ultra rare!"

If cooldown active:
"⏳ Mine ready in 35s."

If no energy:
"⚡ No energy. Come back later or use /mineshop."

If not in room and room required:
"Join the room to mine."

PART 6: MINING XP AND LEVELS
==================================================

Mining XP is called MXP.

MXP rewards:
- Common drop: 5 to 15 MXP
- Uncommon: 15 to 30 MXP
- Rare: 50 to 100 MXP
- Epic: 150 to 300 MXP
- Legendary: 500 to 1,000 MXP
- Mythic: 2,500 MXP
- Ultra rare: 10,000 MXP

Level formula:
- XP required = level * level * 100

Example:
- Level 1 -> 100 MXP needed
- Level 10 -> 10,000 MXP needed
- Level 50 -> 250,000 MXP needed
