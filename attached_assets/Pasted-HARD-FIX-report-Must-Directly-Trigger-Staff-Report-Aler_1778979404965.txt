PART 2 — Add Direct Staff Alert Call After Save
==================================================

Immediately after the report is successfully saved, call the staff alert sender.

Pseudo:

report_text = " ".join(args[1:]).strip()

await send_staff_alert(
    bot=bot,
    category="reports",
    title="📣 New Player Report",
    message=(
        f"📣 New Player Report\n"
        f"From: @{user.username}\n"
        f"Message: {report_text}\n"
        f"Check: !reports"
    ),
    source_user=user,
)

If the staff alert system uses queue:
await queue_staff_alert(...)

If a function already exists, use it.
If no simple function exists, create one helper:

async def send_staff_report_alert(bot, reporter_username, report_text):
    find staff with staffalerts ON and reports ON
    find their DM conversation_id
    send DM through ChillTopiaBot/host only
    return delivered/skipped/failed counts
