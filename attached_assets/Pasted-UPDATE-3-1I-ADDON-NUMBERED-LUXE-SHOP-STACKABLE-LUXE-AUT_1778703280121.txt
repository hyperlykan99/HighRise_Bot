Rules:
- Deduct only actual elapsed time.
- Save remaining seconds on stop/cancel/error.
- If bot restarts, remaining time should persist.
- If auto stops because of an error, save unused time.
- If session completes naturally, remaining time becomes 0.
- Do not consume all time instantly at start.
- Do not lose time if bot crashes.

PART 10 — LUXE AUTO AND NORMAL AUTO DIFFERENCE
==================================================

Normal auto:
- based on base duration + VIP + tools/rods + boosts
- no Luxe time required
- uses !automine / !autofish

Luxe auto:
- uses purchased Luxe time balance
- can run up to remaining Luxe time
- uses same luck stack, tools, rods, potions, boosts, VIP, events
- uses !automine luxe / !autofish luxe

Important:
Luxe auto does not guarantee rare items.
It only provides longer paid auto time.
