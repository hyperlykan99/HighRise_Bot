UPDATE 3.1H — MINING/FISHING COLLECTION BOOK + AUTO SUMMARY DM

Goal:
Add a collection book system for mining and fishing, plus direct-message summaries after auto-mining and auto-fishing finish.

Main features:
1. !collection — overall collection progress
2. !orebook — mining collection book
3. !fishbook — fishing collection book
4. Discovered ores/fish tracking
5. Completion percentage
6. Rare find history
7. Top collectors
8. Auto-mining completion summary DM
9. Auto-fishing completion summary DM

Do not change tokens.
Do not change room ID.
Do not wipe database.
Do not reset balances, VIP, badges, titles, leaderboards, donations, market data, casino data, blackjack shoe, daily streaks, teleports, bot spawns, or player inventories.
Do not change mining odds.
Do not change fishing odds.
Do not change mining reward values.
Do not change fishing reward values.
Do not re-add reward caps.
Do not add reward cap text.
Do not break automining/autofishing.
Do not spam public chat with long summaries.

PART 2 — COLLECTION TRACKING
==================================================

Track discovered mining/fishing items per player.

Mining collection should track:
- ore/item name
- rarity
- first discovered date
- total found count
- best value if available
- last found date

Fishing collection should track:
- fish/item name
- rarity
- first caught date
- total caught count
- best value if available
- last caught date

Important:
- Existing mining/fishing inventory can be used to backfill discovered items if safe.
- Do not wipe existing inventory.
- Do not duplicate inventory.
- Collection book is tracking/discovery, not a replacement for inventory.

PART 4 — UPDATE MINING/FISHING REWARD HOOKS
==================================================

Whenever player mines an item:
- record it in mining collection
- increment count
- update rarity/value
- detect if it is first discovery
- detect if it completes collection milestone

Whenever player catches fish:
- record it in fishing collection
- increment count
- update rarity/value
- detect first discovery
- detect completion milestone

Do not change odds/rewards.
Do not add reward caps.

First discovery message can be short:

📖 New collection find!
@Player discovered Mythril Ore.

or for fish:

📖 New collection catch!
@Player caught Golden Koi.

If too spammy:
- whisper first discoveries instead of public
- public announce only rare/epic/legendary/mythic discoveries
