Poker V2 Fix — Make Auto Card Delivery Use Same Logic As !hand

Current GitHub review:
- modules/poker_v2.py deals cards into memory correctly:
  player["cards"] = [deck.pop(), deck.pop()]
- !hand works because it reads from in-memory player["cards"].
- Auto card whisper fails because _start_hand() uses:
  asyncio.create_task(_send_card_whisper_best_effort(...))
- That background task is not awaited, does not retry, does not update card_delivery_log, and does not confirm delivery before turn starts.

Goal:
Make auto card delivery reliable by using the same memory card source as !hand and awaiting card whispers sequentially before announcing first turn.

Do NOT modify:
- blackjack
- jail
- Luxe Tickets
- tips
- workflows
- old poker.py except route conflicts
- other games

Part 1 — Replace background card whisper tasks

In modules/poker_v2.py, find _start_hand().

Remove this:

for u in seated:
    asyncio.create_task(_send_card_whisper_best_effort(bot, _T["players"][u]))

Replace with:

delivery_success = 0
delivery_failed = []

for u in seated:
    p = _T["players"][u]
    ok = await _send_card_whisper_confirmed(bot, p)
    if ok:
        delivery_success += 1
    else:
        delivery_failed.append(u)
    await asyncio.sleep(0.7)

Do not announce first turn until this loop finishes.

Part 2 — Create confirmed whisper function

Create:

async def _send_card_whisper_confirmed(bot, player):
    username = player["username"]
    cards = player.get("cards", [])

    if len(cards) != 2:
        _T["card_delivery_log"][username] = {
            "cards": False,
            "whisper": False,
            "reason": "cards_missing"
        }
        print(f"[POKER V2 CARD SEND] @{username} failed cards_missing")
        return False

    msg = f"Your cards: {_fcs(cards)}"

    ok = await _w(bot, player["user_id"], msg)

    if not ok:
        await asyncio.sleep(1)
        ok = await _w(bot, player["user_id"], msg)

    _T["card_delivery_log"][username] = {
        "cards": True,
        "whisper": bool(ok),
        "reason": "" if ok else "whisper_failed"
    }

    print(f"[POKER V2 CARD SEND] @{username} cards=true whisper={ok}")
    return bool(ok)

Important:
- Use _w(), the same safe whisper helper.
- Use player["cards"], the same source as !hand.
- Retry once.
- Update _T["card_delivery_log"].

Part 4 — First actor turn whisper should also be awaited once

After public first turn, send the first actor turn whisper with await instead of background task:

Create if needed:

async def _send_turn_whisper_confirmed(bot, username, player):
    cards = player.get("cards", [])
    if len(cards) != 2:
        return False
    owe = max(0, _T["current_bet"] - player.get("current_bet", 0))
    if owe > 0:
        msg = f"Your turn.\nYour cards: {_fcs(cards)}\nPot: {_T['pot']:,} coins\nCall: {owe:,} coins"
    else:
        msg = f"Your turn.\nYour cards: {_fcs(cards)}\nPot: {_T['pot']:,} coins"
    ok = await _w(bot, player["user_id"], msg)
    if not ok:
        await asyncio.sleep(1)
        ok = await _w(bot, player["user_id"], msg)
    print(f"[POKER V2 TURN WHISPER] @{username} ok={ok}")
    return bool(ok)
