PART 2 — STANDARD RESULT DETAILS
==================================================

Every rare find/catch announcement must include actual result details:

Mining:
- player name
- rarity
- ore name
- weight
- value
- MXP

Fishing:
- player name
- rarity
- fish name
- weight
- value
- FXP

Mining format:

⛏️ BIG FIND!
@Player mined [RARITY] [Ore Name]
⚖️ [weight]kg | 💰 [value]c | ⭐ +[mxp] MXP

Fishing format:

🎣 BIG CATCH!
@Player caught [RARITY] [Fish Name]
⚖️ [weight]lb | 💰 [value]c | ⭐ +[fxp] FXP

If too long, split into two messages:

Message 1:
⛏️ BIG FIND!
@Player mined [RARITY] [Ore Name]

Message 2:
⚖️ [weight]kg | 💰 [value]c | ⭐ +[mxp] MXP

or:

Message 1:

@Player caught [RARITY] [Fish Name]

Message 2:
⚖️ [weight]lb | 💰 [value]c | ⭐ +[fxp] FXP
