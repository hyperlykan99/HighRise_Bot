- auto game loop
- auto event loop
- event end timer
- BJ countdown timer
- BJ turn timer

- RBJ turn timer
- any poker timers if poker exists

4. Clear only temporary in-memory caches:
- cooldown caches if safe
- temporary search results
- temporary confirmation codes that expired
- stale room user cache if it can rebuild
Do NOT clear active player data.

5. Reload settings from SQLite:
- bank settings
- casino settings
- event settings
- auto game settings
- role settings
- gold settings

6. Check expired events and clear them if needed.

7. Recover active BJ/RBJ state from SQLite:
- if BJ/RBJ was active, restore the same table
- if RBJ was active, restore the same shoe_json
- restart remaining turn/lobby timers if needed

8. Restart auto loops if enabled:
- auto mini games
- auto events

"✅ Soft restart complete."
