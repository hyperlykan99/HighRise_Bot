Part 1 — Rejoin check

In the user join handler, add a safe rejoin check.

When any user joins:
1. Check if user has an active jail sentence.
2. If no active sentence, do nothing.
3. If active sentence exists and end_ts > now:
   - teleport user to saved spot "jail"
   - whisper remaining time
   - KeanuShield should handle the jail message if possible
4. If active sentence exists but end_ts <= now:
   - mark sentence expired
   - do not send user to jail

Important:
Do not reset the timer.
Use the existing end_ts from jail_sentences.
