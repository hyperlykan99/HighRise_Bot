Add Fishing Bot System Similar to Mining Bot

Focus only on adding a new Fishing Bot system.
Do not change bot tokens.
Do not change room ID.

MasterAngler

If the project already has a bot role system, add FishingBot/MasterAngler as a new bot role/owner.

==================================================

Fishing should copy the structure of Mining, but stay separate.

Mining uses:
- ores
- ore rarities
- ore weights
- ore prices
- MXP
- mining level
- mining cooldown
- ore inventory
- mining announcements

Fishing should use:
- fish
- fish rarities
- fish weights
- fish prices
- FXP
- fishing level
- fishing cooldown
- fish inventory
- fishing announcements

Do not mix Mining EXP with Fishing EXP.
Do not mix ore inventory with fish inventory.

PART 6 — FISH CHANCES AND VALUES
==================================================

Keep fishing economy fair.
Do not make common fish too profitable.
Do not make Exotic common.

Suggested rarity chance style:
Common = common
Rare = frequent
Epic = uncommon
Legendary = rare
Mythic = very rare
Prismatic = jackpot
Exotic = ultra jackpot

Use a similar chance system to mining if available.

Suggested value ranges after weight:
Common: 10–100 coins
Rare: 150–1,000 coins
Epic: 2,000–10,000 coins
Legendary: 25,000–100,000 coins
Mythic: 100,000–350,000 coins
Prismatic: 300,000–900,000 coins
Exotic: 750,000–2,000,000+ coins

Do not implement a dangerous global economy rescale.
Only set fishing values inside Fishing Bot.

==================================================
PART 7 — FISH WEIGHT SYSTEM
==================================================

Fish should have weights.

Use pounds by default:
lb

Each catch should roll a weight between fish min_weight and max_weight.

Final value formula:
final_value = base_value × weight × event_multiplier

Round final value to clean integer.

Example:
Dragonfish base value 5,000
Weight 12.4 lb
Final value 62,000 coins

Fishing messages should show:
- rarity
- fish name
- weight
- value
- FXP earned

PART 8 — /fish COMMAND
==================================================

/fish should:
1. Check player cooldown.
2. Roll fish rarity.
3. Roll fish from rarity.
4. Roll weight.
5. Calculate value.
6. Add fish to player fish inventory.
7. Add FXP.
8. Apply fishing boosts/events if active.
9. Announce if fish rarity is at or above configured announcement rarity.
10. Show player catch message.

Default cooldown:
30 seconds

Starter rod should default to:
30s catch time/cooldown

Catch message format:

MasterAngler:
🎣 Fishing
You caught [COMMON] 🐟 Minnow | ⚖️ 0.8lb | 💰 12c | FXP +3

Legendary catch:

MasterAngler:

Prismatic catch:

MasterAngler:
🎣 Fishing
You caught [PRISMATIC] 🌈 Aurora Koi | ⚖️ 18.2lb | 💰 75,000c | FXP +900

If using color:
- [PRISMATIC] rainbow
- fish name pink for prismatic catches
- always close colors with <#FFFFFF>
