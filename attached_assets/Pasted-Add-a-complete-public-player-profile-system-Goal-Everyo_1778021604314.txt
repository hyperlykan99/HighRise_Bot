Add a complete public player profile system.

Goal:
Everyone can check anyone’s profile, but players can choose whether to show private info like coin balance.

IMPORTANT:
- Do not remove existing features.
- Keep coins, XP, bank, shop, badges, titles, VIP, casino, BJ, RBJ, poker, quests, achievements, reputation, gold, reports, notifications, and staff roles.
- Use SQLite.
- Use get_display_name(username).
- Keep every Highrise message under 250 characters.
- Split long profiles into pages.
- Prevent crashes from bad inputs.
- Add explicit command routing before unknown-command fallback.

==================================================
PROFILE COMMANDS
==================================================

Public commands:
- /profile
- /profile <username>
- /pinfo <username>
- /whois <username>
- /stats
- /stats <username>
- /badges <username>
- /titles <username>
- /achievements <username>
- /casino <username>
- /privacy
- /privacy money on/off
- /privacy casino on/off
- /privacy achievements on/off
- /privacy inventory on/off

Aliases:
- /me = /profile
- /whois <username> = /profile <username>
- /pinfo <username> = /profile <username>

Staff commands:
- /profileadmin <username>
- /profileprivacy <username>
- /resetprofileprivacy <username>

Permissions:
- Everyone can view public profile info.
- Players control their own privacy settings.
- Owner/admin can view full staff profile details using /profileadmin.
- Owner/admin can reset privacy settings only if needed.

==================================================
PRIVACY SETTINGS
==================================================

Create profile_privacy table:

profile_privacy:
- username TEXT PRIMARY KEY
- show_money INTEGER DEFAULT 1
- show_casino INTEGER DEFAULT 1
- show_achievements INTEGER DEFAULT 1
- show_inventory INTEGER DEFAULT 1
- updated_at TEXT

Commands:

/privacy
Shows:
"🔒 Privacy
Money ON | Casino ON
Achievements ON | Inventory ON"

/privacy money off
Hides coin balance/economy from public profile.

/privacy money on
Shows coin balance/economy.

/privacy casino off/on
Toggles casino stats.

/privacy achievements off/on
Toggles achievements.

/privacy inventory off/on
Toggles badges/titles inventory.

Messages:
"✅ Money privacy OFF. Coins are hidden."
"✅ Casino stats are now visible."
"Use /privacy money on/off."

Rules:
- Privacy affects public viewers.
- Player can always see their own full profile.
- Owner/admin can view full profile with /profileadmin.
- Staff should not expose hidden info publicly.
