Fix Poker Turn Sequence — Public Turn Message + Private Card Whisper

Current issue:
Poker public chat currently says:
👉 YOUR TURN @Player | POT:200 | !check !r !fold !allin

I do NOT want that public message.

Wanted behavior:
Public room message:
⏳ @Player’s turn. 30s

Private whisper to current player only:
🎯 Your turn. Cards: [card1] [card2]
Pot: 200 🪙
Use !check, !call, !raise [amount], !fold, or !allin.

Also:
First hand still does not show first player’s hand in whisper.

Goal:
1. Public chat only shows whose turn it is + timer.
2. Current player gets their cards by whisper.
3. First player after blinds gets a guaranteed card whisper.
4. Other players do not see command list spam.
5. Other players can see whose turn it is.

Part 2 — Add private current-player whisper

Whenever a turn starts, whisper current player:

🎯 Your turn. Cards: [card1] [card2]
Pot: [pot] 🪙
Use !check, !call, !raise [amount], !fold, or !allin.

If message too long, use:

This whisper should be sent to:
current_turn_user_id only

Part 3 — Guaranteed first player card whisper

After blinds are posted and first actor is chosen:

1. Announce blinds publicly:
🃏 Dealer: Hand #[n]
🟡 Blinds SB:@SmallBlind ([amount] 🪙) BB:@BigBlind ([amount] 🪙)
POT:[amount] 🪙

2. Wait 5 seconds:
await asyncio.sleep(5)

3. Set current player / first actor.

4. Whisper first actor:
🎯 Your turn. Cards: [card1] [card2]
Pot: [pot] 🪙
Use !check, !call, !raise [amount], !fold, or !allin.

5. Public chat:
⏳ @FirstActor’s turn. 30s

6. Start turn timer after the whisper is sent.

Important:
Do not allow first actor action before cards whisper is sent.
If action comes too early:
⏳ Dealer is preparing your turn. Please wait.

Part 4 — Send cards to all players at hand start, but resend current actor

At hand start:
- deal hole cards to everyone
- save cards
- attempt to whisper cards to all players

But for the first actor:
- after the 5-second blind delay, whisper their cards again as guaranteed delivery.

This fixes first player missing hand.

Part 5 — Card lookup must use same stored state

Make sure the whisper uses the actual stored hole cards.

Do not create new cards.
Do not redeal.

Use existing table/player hand state:
player.hand
player.cards
table.hole_cards[user_id]
or whatever the poker module already uses.
