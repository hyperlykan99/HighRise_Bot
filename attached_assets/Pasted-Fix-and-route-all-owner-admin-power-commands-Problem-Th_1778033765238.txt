PART 2: FIX COIN COMMANDS
==================================================

Add/fix:
- /addcoins <username> <amount>
- /removecoins <username> <amount>
- /setcoins <username> <amount>
- /editcoins <username> <amount>
- /resetcoins <username>

Behavior:
- /addcoins adds to current balance.
- /removecoins removes but never below 0.
- /setcoins sets exact balance.
- /editcoins is alias for /setcoins.
- /resetcoins sets balance to 0.

Examples:
- /setcoins @4ktreymarion 10000000
- /addcoins angelkisses18 5000
- /removecoins user 1000

Messages:
"✅ @user coins set to 10,000,000c / 10M."
"✅ Added 5,000c / 5K to @user."
"✅ Removed 1,000c / 1K from @user."

If usage wrong:
"Use /setcoins <user> <amount>."
