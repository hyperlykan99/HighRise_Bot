==================================================

==================================================
COMMAND MEANING
==================================================

!ptwallet [amount]
Sets the ChillTopiaMC Party Tip Wallet amount.

Example:
!ptwallet 300

Response:
🎉 Party Wallet Set
Wallet: 300g
Handled by: ChillTopiaMC
BankerBot affected: NO

!ptadd [user] [minutes]
Adds a temporary Party Tipper.

Example:
!ptadd @Player 60

Response:
✅ Party Tipper Added
User: @Player
Duration: 60m
Can use: !tip only
Party Mode Required: YES

!ptremove [user]
Removes one Party Tipper.

Example:
!ptremove @Player

Response:
✅ Party Tipper Removed
User: @Player

!ptclear
Removes all temporary Party Tippers.

Response:
✅ All Party Tippers Cleared

!ptlist
Shows current Party Tippers.

Response:
🎉 Party Tippers
@Player — expires in 42m — used 25g/100g

!ptlimits
Shows Party Tip limits.

Response:
🎉 Party Tip Limits
Single: 10g
All: 5g each
Daily: 100g
Wallet Max: 500g
Party Mode Required: YES

!ptlimit [single|all|daily|max] [amount]
Changes Party Tip limits.

Examples:
!ptlimit single 10
!ptlimit all 5
!ptlimit daily 100
!ptlimit max 500

Response:
✅ Party Tip Limit Updated
single = 10g
