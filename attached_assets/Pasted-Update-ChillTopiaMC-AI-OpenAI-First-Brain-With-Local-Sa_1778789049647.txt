     "intent": "...",
     "command": "...",
     "args": [],
     "risk": "low|medium|high|blocked",

     "reply": "short user-facing reply"
   }
8. Parse JSON safely.
9. Validate command against whitelist.
10. Check role permission.
11. If command is safe direct, execute existing command handler.
12. If risky, prepare confirmation.
13. If answer/general question, charge Luxe Tickets if needed, then answer.
14. If blocked, refuse.
15. Never expose secrets or private data.
