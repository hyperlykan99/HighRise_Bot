Type: !confirmbuycoins
Cancel: !cancelbuycoins

Store pending confirmation for 60 seconds:

pending_buycoins[user_id] = {
  spend_tickets,
  receive_coins,
  packs,
  created_at,
  expires_at
}

!confirmbuycoins:
- re-check pending exists
- re-check pending is not expired
- re-check player still has enough Luxe Tickets
- re-check pack calculation
- deduct Luxe Tickets
- add ChillCoins
- log transaction
- clear pending confirmation

!cancelbuycoins:
✅ Coin purchase cancelled.

If expired:
⚠️ Confirmation expired. Use !buycoins max again.
