==================================================
PART 1 — Texas Hold’em Player Count
==================================================

Poker V2 is Texas Hold’em for 2–6 players.

Rules:
- Minimum players: 2
- Maximum players: 6
- 2-player game is still Texas Hold’em, but uses special 2-player button/blind order.
- 3–6 players use normal Hold’em order.

Do not call this “heads-up poker mode” in player messages.
Use normal poker messages.

==================================================
PART 2 — Correct Dealer / Blind Rules
==================================================

For 2 players:
- Dealer/Button is Small Blind.
- Other player is Big Blind.
- Preflop: Small Blind acts first.
- Flop/Turn/River: Big Blind acts first.

For 3–6 players:
- Player left of Dealer is Small Blind.
- Next player is Big Blind.
- Preflop: first active player left of Big Blind acts first.
- Flop/Turn/River: first active player left of Dealer acts first.

Add helper functions:

get_active_seats()
get_next_active_after(index)
get_preflop_first_actor()
get_postflop_first_actor()

Do not hardcode only 2 players.

Add logs:

[POKER V2 TURN ORDER] players=2 street=flop first=@Player reason=bb_first_postflop
[POKER V2 TURN ORDER] players=4 street=preflop first=@Player reason=left_of_bb
[POKER V2 TURN ORDER] players=4 street=flop first=@Player reason=left_of_button

PART 3 — Fix Betting Resolver
==================================================

The resolver must decide the next step after every action:
- check
- call
- raise
- fold
- allin
- timeout auto-check
- timeout auto-fold
- leave fold

Create or fix one function:

resolve_betting_state(bot)

Resolver order must be:

1. If hand_ended or cleanup_in_progress:
   return immediately.

2. Get eligible players:
   players with status active or allin.

3. Get non-folded players:
   players with status active or allin.

4. If only one non-folded player remains:
   award pot immediately.
   complete_hand(reason="fold_win")
   return.
   Do NOT run board.
   Do NOT showdown.

5. If fewer than 2 eligible players remain:
   award/cancel safely.
   complete_hand(reason="not_enough_eligible")
   return.

6. If nobody can act anymore:
   - If board has fewer than 5 cards:
       run remaining board to showdown.
   - Else:
       showdown.
   return.

7. If current betting round is complete:
   advance to next street.
   return.

8. Otherwise:
   move to next player who needs action.
   return.

==================================================
PART 4 — Define “Needs Action” Correctly
==================================================

A player needs action if:

status == active
AND stack > 0
AND (
    acted == False
    OR current_bet < table.current_bet
)

A player does NOT need action if:
- folded
- allin
- has matched current_bet and acted
- has no stack left

Function:

needs_action_players()

Important:
After Big Blind checks preflop:
- SB already acted/called
- BB acted/checks
- all active players matched current_bet
- no one needs action
- therefore advance to flop

Do not ask BB again preflop.

==================================================
PART 5 — Define “Betting Round Complete”
==================================================

Betting round is complete when:

For every active player:
- acted == True
- current_bet == table.current_bet

All-in players are ignored for “needs action” because they cannot act.

If betting round complete:
- preflop -> deal flop
- flop -> deal turn
- turn -> deal river
- river -> showdown

PART 7 — Fix All-In + Call Bug
==================================================

Bug:
Player A goes all-in.
Player B calls.
Bot still asks for turns.

Correct behavior:
After all-in is called, if no active player needs action:
- immediately run remaining board to showdown
- do not ask anyone for another turn

Example:
🔥 @PlayerA is all-in for 4,900 coins.
💰 @PlayerB called 4,900 coins.
🔥 All-in called. Running the board...
🃏 Flop: A♠ K♦ 7♣
🃏 Turn: 10♥
🃏 River: 2♣
👀 Showdown

For 3–6 players:
If Player A all-in and Player B calls, but Player C still needs action:
- continue to Player C
- do not run board yet

Only run board when every remaining active player has matched/folded/all-in and nobody needs action.

==================================================
PART 8 — Board Run Rules
==================================================

Only run remaining board if:
- at least 2 eligible players remain
- nobody needs action
- at least one player is all-in OR no active player can act

Do NOT run board if:
- only one player remains
- everyone else folded
- hand_ended == True
- cleanup_in_progress == True

Before run board:
check len(eligible) >= 2

If not:
complete_hand(reason="fold_win")
return

==================================================
PART 9 — Street Advancement
==================================================

When advancing street:

Preflop -> Flop:
- burn one card
- deal 3 flop cards
- phase = flop
- reset current_bet = 0
- reset each active player current_bet = 0
- reset acted = False for active players
- public:
🃏 Flop: A♠ K♦ 7♣
Pot: 200 coins
- set turn to postflop first actor

Flop -> Turn:
- burn one card
- deal 1 turn card
- phase = turn
- reset bets/acted
- public:
🃏 Turn: 10♥
Board: A♠ K♦ 7♣ 10♥
- set turn to postflop first actor

Turn -> River:
- burn one card
- deal 1 river card
- phase = river
- reset bets/acted
- public:
🃏 River: 2♣
Board: A♠ K♦ 7♣ 10♥ 2♣
- set turn to postflop first actor

River complete -> Showdown.
